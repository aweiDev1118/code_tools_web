# 2026-03-07 v2.0 Phase 1 + Phase 2 功能开发

## 概述

本次迭代为 DevToolbox 新增 8 个开发工具 + 全局命令面板，并完善 i18n 多语言支持。所有功能为纯前端实现，无后端依赖。

## 新增工具

### Phase 1（4 个工具 + 命令面板）

| 工具 | 文件 | 分类 | 说明 |
|------|------|------|------|
| JSON ↔ YAML | `JsonYaml.vue` | 编码转换 | JSON 与 YAML 双向实时转换，使用 js-yaml 库 |
| AES 加密解密 | `AesEncrypt.vue` | 加密解密 | 支持 CBC/ECB/CTR 模式，Base64/Hex 输出，使用 crypto-js |
| CSV/TSV 工具 | `CsvTool.vue` | 文本工具 | CSV/TSV 解析、表格预览、格式转换 |
| SQL 格式化 | `SqlFormatter.vue` | 代码工具 | SQL 美化/压缩，支持 MySQL/PostgreSQL 方言，自定义缩进 |
| Cmd+K 命令面板 | `CommandPalette.vue` | 全局组件 | 快捷键搜索跳转工具，支持最近使用排序 |

### Phase 2（4 个工具）

| 工具 | 文件 | 分类 | 说明 |
|------|------|------|------|
| RSA 密钥生成 | `RsaKeyGenerator.vue` | 加密解密 | Web Crypto API 生成 RSA 密钥对，支持 2048/4096 位 |
| 调色板生成 | `ColorPalette.vue` | 颜色工具 | HSL 算法生成 10 色阶 + 互补色，支持颜色拾取 |
| JSON Schema | `JsonSchema.vue` | 代码工具 | 从 JSON 自动推导 JSON Schema Draft-07 定义 |
| Emoji 选择器 | `EmojiPicker.vue` | 生成器 | 8 大分类、中英文搜索、最近使用历史、一键复制 |

## 技术要点

- **AES 加密**: 使用 `crypto-js` 库，支持 AES-128/192/256，PKCS7 填充
- **RSA 密钥生成**: 使用浏览器原生 Web Crypto API，无外部依赖，密钥不上传服务器
- **SQL 格式化**: 纯前端自实现 tokenizer + formatter，无第三方 SQL 解析库
- **调色板算法**: 基于 HSL 色彩空间，通过调整明度/饱和度生成色阶，色相偏移 180° 生成互补色
- **命令面板**: 支持模糊搜索（id/name/description/keywords），最近使用优先排序
